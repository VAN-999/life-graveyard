# ☠️ 人生数据坟场

> 把你的每一天，活成一座墓园

---

## 这是什么

一个记录你数字生命的后端项目。

你每天走多少步、看了多久手机、敲了多少次键盘——这些数据都会被记录下来，变成一份荒诞的"死亡报告"。

---

## 玩法

**1. 注册账号** → 给自己立块碑，默认送一块极简白碑。

**2. 提交数据** → 每天提交步数、屏幕使用时间、键盘敲击量、睡眠时长等，你的数据越离谱，报告越精彩。

**3. 每日死亡报告** → 每晚零点自动生成：
- 今日死亡原因（例：“死于连续久坐6小时”）
- 阴阳怪气墓志铭
- 今日死亡指数评分

**4. 装修墓场** → 做任务赚冥币，买装饰品：
- 墓碑：极简/花岗岩/赛博霓虹/代码碑
- 祭品：电子菊花、电子啤酒、电子MacBook
- 宠物：赛博乌鸦、像素幽灵、代码萤火虫
- 特效：暴雨、极光、代码雨

**5. 阴间任务** → 完成离谱任务领奖励：
如：
- “屏幕时间超过10小时” → 奖励电子蜡烛
- “凌晨3点访问墓场” → 奖励月光滤镜
- “连续3天不刷朋友圈” → 奖励僵尸皮肤

**6. 社交** → 加好友、送电子菊花、在悼念簿留言。在线状态显示“诈尸中”或“安息中”。

**7. 废墟+盗墓** → 好友30天不登录，墓场变废墟，你可以去偷他的装饰品，成功率70%，失败会被诅咒（自己丢一件）。

**8. 轮回转世** → 累计30份死亡报告后可转世，数据重置，保留部分装饰品，送你一块“轮回纪念碑”。
